To compile the project, add JARS to this lib folder:

• ojdbc6.jar (oracle.com)
• The OpenJump JAR found in the lib folder of an OJ Installation: OpenJUMP-1.8.0-r4290-nosrc.jar for instance
• The JTS JAR found in the lib folder of an OJ Installation: : jts-1.14.jar for instance
• The gt2-oracle-spatial found in the lib/ext/dbquery_plugin folder of an OJ Plus installation: : gt2-oracle-spatial-2.3.2.jar
