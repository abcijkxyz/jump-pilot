INSTALLATION

Copy the whole folder contents into your openjump folder. This is the folder 
holding the folders /bin and /lib. 
Start openjump.


DEINSTALLATION

Remove all files you find in the distribution zip file from your openjump 
folder.


REINSTALLATION

Please do a deinstallation as described above.
Then procede as described in the new distributions Installation 
steps text file.

SOURCES

Find the sources on sourceforge
http://jump-pilot.svn.sourceforge.net/viewvc/jump-pilot/plug-ins/de.soldin.jump/
or in the respective extension jar file
<jumphome>/lib/ext/<ext_id>.jar
